Tim Abbott (Apr 20 2020 at 17:09):

Back to the original alignment question, the UI model of right-aligning messages you yourself sent opposite messages other sent works well for a mobile-only chat tool designed for small groups where the screen is small and you send a significant portion of messages, but it's a pretty bad experience in a desktop size browser (where the right-alignment would force one's eyes to scan pretty far away) as well as being the wrong focus for a stream with dozens of users or more (where messages you yourself sent are rare and would just disrupt reading). This is why all of our peer team chat tools use the same model as Zulip with the same alignment for all messages.

I think there is a possible feature of some sort of visual adornment of messages that you yourself sent to make it easy to visually filter those in PM conversations.
