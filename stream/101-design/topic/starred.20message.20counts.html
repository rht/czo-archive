Topic: starred message counts

Tim Abbott (Aug 21 2018 at 21:50):

We just added (only enabled on chat.zulip.org and development environments) a new display setting, starred_message_counts, which lets you get unread-count style display of how many starred messages you have in the web UI, like this:
pasted image

The intent of this feature is to make Zulip better for folks who use starred messages to mean "things I should come back to later". I'd love it if a few people who use starred messages that way (or want to) tried this out, and commented on their experience. One detail that I'm particularly uncertain about is whether we want to display the starred message counts with different styling from normal unread counts; my feeling is currently yes.
