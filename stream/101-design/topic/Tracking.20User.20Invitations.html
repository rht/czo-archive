Topic: Tracking User Invitations

Jore (Aug 01 2021 at 06:19):

Hi all. Is there a way for all to see which users were invited by whom after they've accepted the invite? Currently, I can see pending invitations that show whom invited whom at Settings -> Organisation -> Invitations but after a user is added, it appears the only record of this is transaction is stored in a Private Message -> Notification Bot which means nobody else in the Zulip instance gets accountability around whom is adding whom. I guess I'm proposing that this information be added to the Core Team stream, much in the same way that new users are added to the signups topic there. For example, perhaps Bob just signed up for Zulip... could be changed to Bob accepted Alice's invite to join Zulip or something, if that is the case. Measures like this are important for our team where our instance is Invite only, and admins/moderators need to be keeping an eye on who has been invited, when, and by whom.
