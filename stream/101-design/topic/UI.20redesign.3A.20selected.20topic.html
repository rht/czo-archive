Vlad Korobov (Jan 23 2022 at 22:33):

Hey guys, here is the update. I suggest to have this design for the left panel:

  1. Selected narrower has not a white background but a bit lighter grey. This allows to move from tinted blue hover color to have 2
  2. White background for the hover. Provides maximum contrast.
  3. No rectangles around counters allows to increase number size, increase horizontal space for the text. Right now counters grab too much of attention on a desktop. But if you don't trust my vision here and want to increase attention, we can do is to make them a bit darker/bolder.
  4. We still need to experiment with hiding counter number on hover, but if we don't like it I suggest to shift a number on hover to the left to free up space for the context button.
