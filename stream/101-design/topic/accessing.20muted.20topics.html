RDT (Oct 04 2020 at 10:05):

with time user will forget which topics the user has muted, and if due to some reason the user needs to see some muted topics, it's a long process of going to settings and viewing muted topics there, which I think user will forget.

but I don't think a new user of Zulip will know this. Even I didn't knew where to see all my muted topics till I opened settings for changing my profile pic.
