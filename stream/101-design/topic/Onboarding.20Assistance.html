Armaan Ahluwalia (Jul 02 2018 at 18:42):

I believe it could be useful to improve the onboarding tips in the following way:

Instead of showing all the tips at the moment the user starts using the app I believe we could be more intelligent about when we show a particular tip. For example, we could set certain conditions in order for a tip to be displayed -
If a user just started using the app we could stagger the tips in a smart way. Like maybe show them 1 or at most 2 tips at the very beginning and then slowly introducing increasingly advanced features over time.

I've been using a couple of apps that do this and its actually really nice because it doesn't feel like you're bombarded with hints at the very beginning and also gives you time to get to grips with the UI before introducing more advanced ideas.
