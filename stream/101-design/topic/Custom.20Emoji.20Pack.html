Topic: Custom Emoji Pack

Mehdi Hamidi (Sep 19 2021 at 12:06):

Hey Guys.
Is it possible to add custom emoji pack (i.e. emoji font) in Zulip?
My intention is specially using JoyPixels emojis in Zulip.
Is there any document that I can follow and build my own Zulip server with custom emoji pack included?

Anders Kaseorg (Sep 19 2021 at 20:58):

No, we only support the freely licensed emoji sets provided through emoji-datasource. You can patch Zulip however you’d like if you have the technical expertise, but there’s no pre-baked option or documentation for this particular customization, and this is unlikely to change because we can’t distribute Zulip with non-free emoji sets.
