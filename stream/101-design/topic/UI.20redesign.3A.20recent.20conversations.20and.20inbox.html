Tim Abbott (Jun 27 2022 at 22:46):

Hmm, what's the thinking behind using a bold font for the stream names in these prototypes? We had previous versions of Zulip where we used bold for the stream name where it appeared in the recipient bars in the message feed, and it was not popular because it drew the user's eye to the stream name, rather than the messages themselves. I think that is less of a valid criticism of using it here, but I'd still argue that the topic is probably more important information than the stream name in the recent conversations view, if we're going to emphasize something, but that it might look cleaner with a consistent font between the two, since that's what we're generally doing elsewhere? (E.g. in message recipient bars).

Tim Abbott (Jun 27 2022 at 22:49):

I think a core design tension for "recent conversations" is that on the one hand, we kinda want to display the stream/topic pairs like a message recipient bar for design consistency reasons, but on the other hand, the table design is ergonomic for sorting, filtering, and also just separating the streams from the topics for easier visual scanning.

Tim Abbott (Jun 27 2022 at 22:50):

(I definitely agree with the strategy of having these be separate views, as it'll make several things simpler, so thanks for pushing for that direction!)
