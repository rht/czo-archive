Tim Abbott (Jan 06 2023 at 02:11):

The only other thing I'm worried about with the closed compose box experience is whether will be sad that there's no way to make a new private message directly in this redesign.

I suspect we will find that some users will have weird adaptations like clicking on random users in the right sidebar and then changing the recipient of the compose box that gets opened; that might feel like less work than opening the compose box for a stream and then changing to a private message recipient type.

I suppose if we hear complaints, we can add something inside a \vdots menu at the far right or something similarly subtle.

I don't see a cheap way to figure out whether this will prove to be a real issue. I personally use the x keyboard shortcut to start new private messages while looking at a stream a few times a week, but anecdotally, I am not sure I know of anyone who regularly clicks the current "New private message" button with equivalent functionality.

Tim Abbott (Jan 06 2023 at 02:12):
