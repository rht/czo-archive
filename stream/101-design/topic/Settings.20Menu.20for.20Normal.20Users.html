Topic: Settings Menu for Normal Users

Armaan Ahluwalia (Nov 21 2018 at 14:39):

Does it make sense to show "normal" users a stripped down version of the settings menu. I feel like its confusing for them to see an organization settings option as well as the tab when they open up their user settings. I think a better idea would be to have an option to hide such admin level options from them. Same for the statistics option for example.

Steve Howell (Nov 21 2018 at 17:41):

I don't think we want to completely hide organization settings from normal users, because normal users sometimes want to know what the settings are. I wonder if a less extreme solution is to make it more obvious which settings are personal and can be changed without an admin's help.

Rishi Gupta (Nov 21 2018 at 18:17):

agreed both that we want them to be able to see it, and that currently it is confusing. I think ideally when you switch to Organization settings there would be a (constant) banner at the top or bottom that explains these are for viewing pleasure only.

Armaan Ahluwalia (Nov 21 2018 at 19:46):

I think ideally, as a regular non admin zulip user, I would want to be able to see the settings ( in a read only mode ) as some sort of an “advanced” type of access feature. My feeling is the drop down menu is way too cluttered as is and feels overwhelming as a normal new user.

Tim Abbott (Nov 27 2018 at 00:52):
