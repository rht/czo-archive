Vlad Korobov (May 30 2023 at 20:49):

So the problem with highlighted message is that background color applied on the very top level:
image.png
So if we mask all that, we are masking focus rect too. To do that trick with a mask we have to change what divs are in which. That is why I told @Alya Abbott that is not a simple task.

Ideally is to have a message container div which doesn't do anything, while inside have layers:
Layer - 0 -highlighting+hover
Layer - 1 - User,
Layer - 2 - Message content
Layer - 3 - focus rect, unread marker

Layer 1 and 2 could be together

With this approach we could control things independently. For masking in case of more less we could apply the same mask twice:

  1. for the highlight
  2. for the content.
