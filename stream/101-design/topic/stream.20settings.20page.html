• We would set up the page with the "personal" tab (middle one) active, as I guess this would be the main use case for this page.

• Do we want to extend the tab switches to fill up the entire row?
(Then we could extend the tab names from "General" to "General Settings")
• For not subscribed streams I was just going to disable the "Personal" tab. (make it non-clickable)
• I guess we should find a new place for the 3 buttons group.
Maybe we could place them in a common bottom row? Then for the create new stream page it would look consistent if we decided to add next / back buttons?
