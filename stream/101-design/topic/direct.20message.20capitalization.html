Topic: direct message capitalization

Alya Abbott (Apr 29 2023 at 01:43):

Hey! How should we capitalize "direct message" in the compose box stream picker? The plan (cf. #25116) had been to use "Direct Message". It's currently merged as "Direct message" as a last-minute linter failure fix. But we should decide what we actually want. :)

The idea with "Direct Message" is that it is more consistent with the "DM" abbreviation, which we are using once that option is selected. (And "Dm" would look quite odd.) "Direct message" follows our general sentence case pattern.
