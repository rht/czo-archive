Tim Abbott (May 26 2023 at 16:52):

I think removing it makes sense. I'm pretty sure Ctrl+[ was added along with the j/k Vim keyboard shortcuts were added, very early on. We get a lot of feedback from folks appreciating the j/k support, but I've never heard anyone mention this, and I feel like modern keyboards all have a way to do Esc pretty easily, so I don't think there's much reason to have this.

(Cmd+[ was presumably generated from translating Ctrl to Cmd like we do for Mac shortcuts in general)
