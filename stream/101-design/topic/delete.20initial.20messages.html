Tim Abbott (Sep 12 2017 at 14:36):

Just because I think we want it to be really hard to mass-delete everything.

Tim Abbott (Sep 12 2017 at 14:38):

We should be thoughtful about the specification: I think it should delete (using the same Archive approach we use for other message deletions, so one can recover if it happens accidentally) the messages sent by the human users; presumably we don't want to delete the bot messages. and have a confirmation dialog.
