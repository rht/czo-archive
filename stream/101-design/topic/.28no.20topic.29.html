Could a visual indication that a mentioned user has since been renamed be added? Mentions seem to store the user's ID in addition to their username.

Ugly mock-up of a possible solution: screenshot-2018-04-18-03-13-13.png

Another approach is to simply change the mention text to match the updated username; I know that, for example, Discord does this. The upside is that this retroactively removes offensive or undesirable usernames from chat history; the downside of this approach is that it still destroys context. (Basic illustration: User A jokes about User B's name, User B changes name, backlog no longer makes sense.)

Only listing both the original username and the updated username preserves context entirely, but again, also preserves potentially offensive usernames in chat history. (Which I suppose in certain contexts might also be desirable, e.g. as evidence.)

Tim Abbott (Apr 18 2018 at 16:05):

@Geni Genius for some context, for new users, there's definitely an advantage to having large font sizes (they're certainly popular today), but if you're using the tool every day, you really do want the density provided by our current settings. So our general plan here is to change the default font size, but have a high-density option.
