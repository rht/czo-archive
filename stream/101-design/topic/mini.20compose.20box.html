Abhijeet Bodas (Apr 16 2021 at 12:47):

Is it right that most experienced users use r/enter/clicks to reply to messages?
If so, how about this - we have an actual text box at the bottom, which would be pretty barebones and allow you to send a message only to the topic of the selected message. New users would first see this and immediately know where to type, and old users can still use it to send short and quick messages. And we can make that text area expand into the normal compose box if the message gets too long. And we can (as it is now) have r and enter trigger the normal, full-fledged compose box, to still satisfy advanced users.

Steve Howell (Apr 16 2021 at 14:52):

Mostly, I don't think advanced users actually want this. It's nice to have the compose box closed when you're in reading mode, and if you want to reply, just hit enter, and the compose box is automatically focused at a useful size. Having the compose box open but unfocused has an awkwardness to it.

Beyond that, I just think improving the buttons is a more incremental change here.
