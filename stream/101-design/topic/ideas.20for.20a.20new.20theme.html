Steve Howell (Mar 25 2020 at 10:47):

I'd really like to approach new themes by embracing a Stylish/Stylebot type solution. Folks on the Zulip project can install these plugins and experiment with their own styles changes. If we find places where it's hard to override Zulip styles with Stylish, we can try to make Zulip's scheme easier to overwrite. And then we can publish unofficial themes on some GH repo outside of Zulip.

The idea here is to actually encourage a little bit of "unofficial" customization, sort of similar to what happens for games (where people build their own mods). I'm not really a gamer, so I'm maybe not doing the game analogy justice.
