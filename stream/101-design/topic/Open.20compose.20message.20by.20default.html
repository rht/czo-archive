The main tradeoff with opening the compose box automatically is that keys like n stop working. We've had many iterations of changing this behavior, so I don't we're likely to change it at this point.

I think the reason we say "New topic" is to encourage using that button only when you really are starting a new topic. The rest of the time we want people to be more in reply-to-message mode.
