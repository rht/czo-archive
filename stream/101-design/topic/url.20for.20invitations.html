Topic: url for invitations

Rishi Gupta (Aug 02 2017 at 16:21):

What do we want the URL for invitations and multiuse invitations to be? For single invitations we currently use URLs like example.zulip.com/accounts/do_confirm/13snth3s4nh2sn4h23aoesc9, though I think it makes sense to change these URLs be prettier as well.

Maybe something like
example.zulip.com/join/13snth3s4nh2sn4h23aoesc9 for single use
example.zulip.com/join/user-specified-string-13snth3s4nh2sn4h23aoesc9 for multi use?

I think it'd be nice to have a distinction between the single and multi use URLs, though I guess not necessary.
For user-specified-string: By default (and for V1) I'm imagining no user-specified-string, but eventually I think we will want to allow admins to enter one anyway so that they can keep track of their 3 different multiuse invite links. (Got this idea from our tshirt sourcing company!)
