Topic: default streams

Tim Abbott (May 16 2017 at 23:52):

I'd like to start a conversation about what the default stream name/descriptions should be in a new Zulip organization:
https://github.com/zulip/zulip/issues/4803

Tim Abbott (May 17 2017 at 00:06):

Some thoughts:

  • I like having a "zulip" stream, since it's a convenient place for folks to discuss Zulip and get peer support
  • Some folks have suggested s/announce/announcements/
  • It's hard to name streams without knowing anything about an organization, but I feel like perhaps starting with an "off topic" stream may be helpful for people getting started
