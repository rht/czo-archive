After chatting to @Steve Howell in #feedback > /nick command, I've put together a prototype of having a /theme slash command. I've not adjusted any of the typeaheads, the 'undo' framework or tests yet. The idea was not that we would necessarily remove the current commands like /dark, but have more structure to the slash commands and potentially simplify the typeahead - with the prototype one can do /theme dark or /theme day, for example.
I'm not sure how good the code is, but wanted to post here before looking into this further. I have a WIP commit at https://github.com/neiljp/zulip/commit/765948135e64347ca240b8180aa5468c05741d63
