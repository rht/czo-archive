Hi, I wanted to discuss about the UI of issue #9228. : )
This is about displaying the specific streams which does not match with the 5 stream message notification settings defined by the user in the #settings/notifications page. Few ideas I had in mind are:-

Since the notification settings are basically booleans, I was thinking of adding a cross button, clicking it would mean accepting the global settings for that notification and that stream would get removed.

Also, for organisations with many streams listing many unmatched streams for would be a problem. So we could reuse the popup button beside New stream notifications: text in the #organization/organization-settings page and maybe add the cross button beside each unmatched stream.
This would be done for all the 5 stream message notification settings.

I also had a question about where to place the unmatched notification lines:-

  • below the Change notification settings for individual streams on your Streams page. line that is under the global settings.
  • or at the end of the page where slack puts it.

Also we could just place the stream exceptions buttons beside each checkbox lines. But it seems there are plans to change the layout of the checkboxes on notifications tab to a grid structure (#12182).
