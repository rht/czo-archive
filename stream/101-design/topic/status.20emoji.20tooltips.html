Hey! A little while ago, @Greg Price filed an issue for show a tooltip with the emoji name on status emoji (#19821), and there's an in-progress PR. The main motivation is to make it possible to find out what the emoji is called, e.g. to be able to use it. Note also that Greg left a detailed comment here.

My intuition, perhaps shaped by my experience with emoji statuses in Slack at my previous job, was that the tooltip on the emoji should provide the status text, rather than the emoji name. Note that the status text is also included in the tooltip for the user's name in the left sidebar, but I think it can be a little hard to find there.
