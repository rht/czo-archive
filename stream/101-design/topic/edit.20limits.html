Rishi Gupta (Sep 29 2017 at 04:45):

For this setting: pasted image, what would folks think of replacing the input box with a dropdown with options like {1 minute, 10 minutes, 1 hour, 1 day, 1 week, No limit}?

I feel like the extra configurability of allowing any value here is not that useful, and a dropdown setting with a few straightforward options is much easier to reason about as an admin.

Partly motivated by the fact that we're about to add another setting for limits to topic editing, where I think our default is going to be 1 day, rather than 1 hour (or actually our current default here is 10 minutes). So if the topic setting is in minutes, the default value is going to be a meaningless number to most (1440), and if the topic setting is in hours, you have two similar settings next to each other, one in minutes, and one in hours.
