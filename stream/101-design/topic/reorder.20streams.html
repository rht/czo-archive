Plankter (Jun 03 2021 at 17:54):

The ability to reorder streams would be useful for conferences that use the Zulip chat for Q&As. Often, streams are created for the conference sessions, in which case a chronological order is better suited than an alphabetical one. For now, we deal with this by pre-pending the streams with suitable unicode characters. Being able to control the order via the API would be nice.
