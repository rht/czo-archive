Topic: adding subscribers to private streams

Christoph (Jul 27 2020 at 19:56):

Even as admin, I cannot add subscribers to private streams I'm not myself subscribed to. I can remove everyone, but cannot add anyone. IMO this is inconsistent behaviour, especially for admin accounts, which should have full control over every stream or none at all.

Tim Abbott (Jul 27 2020 at 20:02):

This is consistent behavior, with the rule "Administrators cannot access private message or private stream content unless an existing subscriber adds them".

We are considering adding a feature to allow "Organization owners", a new concept in 3.0, to change the rules for how private streams work; since there's definitely organizations for whom that preference doesn't make sense.
