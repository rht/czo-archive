Tim Abbott (Mar 02 2022 at 22:18):

@The One session expiration is controlled by these settings in /etc/zulip/settings.py:

## Controls whether session cookies expire when the browser closes
SESSION_EXPIRE_AT_BROWSER_CLOSE = False

## Session cookie expiry in seconds after the last page load
SESSION_COOKIE_AGE = 60 * 60 * 24 * 7 * 2  # 2 weeks

Tim Abbott (Mar 02 2022 at 22:19):

We should probably document these somewhere; what did you search for when looking for them?

Tim Abbott (Mar 02 2022 at 22:20):

The only way to edit settings before importing an organization is to hand-edit the JSON files generated by the data export tool, since we try to translate the previous site's settings.

Tim Abbott (Mar 02 2022 at 22:20):

(But you can just block users from joining immediately after the import and then edit settings that way).
