Tim Abbott (May 25 2023 at 19:10):

@Jerry Broughton We generally recommend always upgrading to the point releases; they're designed to be very low risk upgrades and often fix important security or reliability issues.

Tim Abbott (May 25 2023 at 19:10):

You generally don't need to do intermediate upgrades; though see https://zulip.readthedocs.io/en/latest/production/upgrade.html#upgrading-the-operating-system for some cases where that's necessary if you're quite far behind.

Tim Abbott (May 25 2023 at 19:12):

You should post the specific errors you're seeing; it is our goal that there not be random exceptions in production instances.

https://zulip.readthedocs.io/en/latest/production/troubleshooting.html has some advice on how to post logs (E.g. use a code block).
