Topic: Migrating users between auth methods

Pranav Ashok (Aug 07 2023 at 08:14):

Hi all, I am running a self-hosted Zulip instance and was wondering if it is possible to "move" users from the basic email+password authentication to an SSO based authentication, without disturbing the user's history and conversations.

Mateusz Mandera (Aug 07 2023 at 12:34):

@Pranav Ashok Sure, these things are independent. Zulip identifies the user account based on the email address obtained during authentication. So you can switch to any other auth method, and as long as Zulip gets the email address matching the existing account, that's what the user will get logged in as
