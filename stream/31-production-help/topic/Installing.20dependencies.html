I have just switched from windows to ubuntu, although this may not be helpful for everybody but just to inform and maybe helping someone - When I installed all the dependencies using npm install, it stopped and the error was something like unable to parse - Unsupported JSON, the fix was 1. npm clean cache --force, 2. rm -rf node_modules, 3. npm install.
