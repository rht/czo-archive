Topic: upgrade from master to new stable?

Jonas F (Sep 26 2020 at 01:51):

Hi!
I have set up a zulip server on a virtual private server for a small research group. To be able to import a server previously running on zulip.com, I upgraded to master, as recommended here. I successfully imported, and all is running fine.
My question: I installed and upgraded to master on September 11, and version 3.2 was released on September 15. Is it (a) possible (b) recommended to update to the new release from master?
My goal is to have a stable installation that requires as little maintaing as possible.

Anders Kaseorg (Sep 26 2020 at 01:55):

We only support an upgrade from master to a stable release if your master commit is from before the stable release was branched. 3.x was branched from master with the release of 3.0 on July 16. So you’ll need to wait for 4.0.

Jonas F (Sep 26 2020 at 01:58):

Got it, thanks a lot! I'll wait for 4.0 then.
