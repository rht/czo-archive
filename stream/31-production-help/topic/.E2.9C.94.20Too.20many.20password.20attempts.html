Mateusz Mandera (Aug 17 2021 at 16:20):

Sure, opened #19586

@Ronny You can run the relevant bit of code from the management command in manage.py shell:

from zproject.backends import RateLimitedAuthenticationByUsername
username = "user@example.com"
RateLimitedAuthenticationByUsername(username).clear_history()
