Lily Chung (Dec 22 2019 at 23:28):

Hi, I'd like to install Zulip in production but I already have a settings.py file created ahead of time. Is there a good way to install without requiring the --hostname and --email options, since I already have those configured in settings.py? (I'm trying to automate this process, so it's easier if I don't have to parse them back out.)

Anders Kaseorg (Dec 22 2019 at 23:43):

You can install with --no-init-db, then copy settings.py into place and run the two commands it tells you to run at the bottom of the install script.
