@Simon Michalke This bit of code in manage.py shell should do it:

from zerver.actions.user_settings import do_change_user_setting
for u in UserProfile.objects.filter(is_bot=False):  # this query can be narrowed down to a specific organization if desired, etc.
    do_change_user_setting(u, "default_language", "de", acting_user=None)
