Topic: Upgrading directly

The One (Apr 28 2022 at 17:57):

Hello. Currently running version 4.10 of zulip. Is it possible to update directly to the latest version of zulip or will I need to update to each version till reaching the latest stable release?

Alex Vandiver (Apr 28 2022 at 18:00):

@The One: You can upgrade directly to 5.1.

The only other thing to check is if you're running a supported version of the OS -- Zulip 5.x supports Debian 10, ebian 11, and Ubuntu 20.04. If you're not running one of those, see https://zulip.readthedocs.io/en/stable/production/upgrade-or-modify.html#upgrading-the-operating-system first
