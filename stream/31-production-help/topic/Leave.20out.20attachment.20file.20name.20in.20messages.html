Tim Abbott (Oct 18 2022 at 15:09):

If you download the file, it's in most situations nice for the filename to have been preserved.

Tim Abbott (Oct 18 2022 at 15:10):

Like, if I upload zulip-server-5.1.tar.gz, and you download it, you'd probably prefer to download a file called zulip-server-5.1.tar.gz, not abcd1234.tar.gz, or some other random string we might generate that isn't the original filename.

And our Markdown system has you pick a label for a __link__ to the uploaded file, not the file itself; if I type You can download the release [here](.../zulip-server-5.1.tar.gz), then nobody wants clicking that link to give you a file called here.

Tim Abbott (Oct 18 2022 at 15:13):

Users concerned about privacy can of course rename a file on their local system to something generic before uploading them.

Tim Abbott (Oct 18 2022 at 15:17):

I'd be curious whether other chat systems provide a way to choose the name of a file while uploading it; I feel like it's a bit of a UI challenge to do so without complicating/slowing the flow (like having a modal pop up asking you what you want to name the new file you're uploading). I could see having a privacy option to offer such a prompt.
