Alex Vandiver (May 26 2022 at 19:18):

5.x does not support 18.04, so you will need to do this in a few steps:

  1. Upgrade to the latest 3.x release, 3.4
  2. Upgrade to 20.04
  3. Upgrade to the latest 5.x release
